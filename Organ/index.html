Grading Criteria Achieved

The grading elements supported are:

  1. 1. Component passes audio
  2. 2. 1 Effect
  3. 3. 3 Effects
  4. 4. Controllable Effects Send
  5. 5. 4 Effects

Specifications implemented

Component Passes Audio:

All audio is checked for an indicated effect. If no effect is indicated, there is no change to the audio. If a specific effect is indicated, only that effect is applied.

1 Effect: Chorus Effect

3 Effects: Flange Effect and Reverberation

Controllable Effects Sends:

Effect for instrument can be specified in the XML score

4 Effects: Noise Gating.

Prevents sound that falls below a threshold
