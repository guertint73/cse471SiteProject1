Component Details

This component implements a simple wavetable based additive synthesizer. This type of synthesizer creates sound by adding together many sinusoids to form complex tones. My implementation allows users to specify the amplitude of as many harmonics as they desire. To enable crossfading between different sound specifications, CAdditiveInstrument contains two CAdditiveWave components which generate the additive sinusoid for the beginning and and of the note. The output of these two components is mixed linearly based on the time since the start of a note.


For efficiency reasons, CAdditiveWave implements playback via a one-cycle wavetable which dramatically improves performance. This complicates vibrato somewhat, and this feature is implemented via oscillating playback speed and linear interpolation to synthesize new frames as needed. The sound of a note is specified by simply adding two attributes to the note tag. "soundstart" and "soundend" specify the amplitudes of the harmonics at the beginning and end of the note respectively. The values of these attributes is simply a spaces separated list of the amplitudes desired. For examples, see the XML.


Grading Criteria Achieved

The grading elements supported are:

  1. 1. Sinusoid playback on demand
  2. 2. Definition of All harmonics (as many as desired by the user)
  3. 3. Envelope Generation (via simple ASR)
  4. 4. Polyphony
  5. 5. Reson Implementation
  6. 6. Vibrato

Specifications implemented

  • Sinusoid playback on demand:
    • Sequencer correctly creates CAdditiveInstrument objects which create sinusoidal sounds.
  • Definition of All harmonics (as many as desired by the user):
      The user can specify the amplitude of as many harmonics as they want in both the soundstart and soundend attributes. If they wish to only set even or odd harmonics, they can simply set the others to 0.
  • Envelope Generation (via simple ASR):
    • Sounds both fade in and out.
  • Polyphony:
    • Multiple CAdditiveInstruments can be active at a time.
  • Sound cross-fading:
    • CAdditiveInstrument fades between two CAdditiveWave outputs over the course of a note. The sounds of each of these are user specified in the XML.
  • Vibrato:
    • CAdditiveWaves play with vibrato by modulating playback speed and using linear interpolation when necessary to synthesize new frames.
